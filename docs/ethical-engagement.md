# 倫理的な利用継続設計

manapick careerは、読者が自分で比較し、離脱・保留・中止を選べることを優先する。滞在時間やクリック数を増やすために、判断を歪める仕組みは使わない。

## 使用しない設計

- ランダム報酬、ガチャ、ニアミス、外れを惜しませる演出
- 連続利用記録を失う不安、未完了を責める通知
- 根拠のない残席、偽のカウントダウン、常設の「今だけ」
- 費やした時間や金額を理由に継続させる表示
- 個人の不安を煽る適職・採用・年収・将来性の断定
- 広告報酬やEPCによる職業・講座の順位変更

## 使用する設計

- 読了前でも結論が分かる要約回答と目次
- 以前できなかった作業ができたかを本人が確認する上達記録
- 出典、最終確認日、適用範囲、分からないことの明示
- 選択を急がせない中止条件、比較項目、公式確認への導線
- 無料の判断材料を先に提供し、必要な場合だけPR選択肢を表示
- PRを外しても編集上の結論・掲載順・比較結果が変わらない構造

互恵性は「先に役立つ情報を公開する」ことで実現し、返報を迫る文言には使わない。希少性は、公式に確認できる期限・定員・制度条件がある場合だけ、確認日と出典を伴って表示する。
