# manapi商店 商品画像クリアランス

確認日: 2026-07-14（JST）

## 現在確認できた事実

- `public/shop/` の6画像は、姉妹サイト `manapick AI` が使用しているローカル画像資産と同一です。
- Amazonの商品画像URLを埋め込んだものではなく、Amazonの商品画像、価格、在庫、レビューは保存していません。
- 各商品カードには、メーカー名とメーカー公式製品ページへのリンクを表示します。

## 未確認

- 6画像について、メーカーからmanapickへの転載・公衆送信を許可する契約、メール、プレスキット規約などの証跡は、リポジトリ内にも提供資料内にもありません。
- 「公式製品ページに掲載されている」「姉妹サイトで既に使っている」「出典を表示する」ことだけでは、画像利用許諾の証明になりません。

## 公開判定

商品画像の利用許諾証跡を確認するか、権利処理済みの独自画像へ差し替えるまで、今回の変更は公開停止とします。確認できた場合は、対象画像、権利者、許諾範囲、確認日、証跡の保管場所を本書に追記します。
