# manapi商店 商品画像クリアランス

確認日: 2026-07-18（JST）

## 2026-07-18の是正内容

- 許諾証跡を確認できなかったメーカー商品写真6点を、公開対象とGit管理対象から削除しました。
- 代替画像は、manapick編集部が本リポジトリ用に作成した抽象的なSVGイラスト6点です。メーカーのロゴ、商品写真、商品固有の外観を複製していません。
- 各カードの代替テキストと画像クレジットに「編集部作成イメージ」「実際の商品写真ではない／実物と異なる」と表示します。
- Amazonの商品画像、価格、在庫、レビューは保存・表示しません。製品仕様はメーカー公式ページへのリンクと独自要約で確認できる構成です。

## 確認方法

- `content/shop-products.ts` の公開画像参照がすべて `/shop/*-illustration.svg` であることを `scripts/check-shop.mjs` で検査します。
- 同検査は、各画像クレジットに編集部作成と実物との差異の明示があることも必須とします。
- 旧WebPのファイル名をGit追跡対象に残しません。

## 公開判定

商品写真の権利未確認を理由とする公開停止条件は、独自イラストへの差し替えにより解消しました。将来メーカー画像へ戻す場合は、対象画像、権利者、許諾範囲、確認日、証跡の保管場所を本書へ追記し、`scripts/check-shop.mjs` の判定を人手確認付きで変更するまで公開しません。
